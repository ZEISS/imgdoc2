EnumerateCoordinatesInBounds(IReadOnlyList<(Dimension Dimension, int Start, int End)>)

Enumerates tile-coordinates as defined by the ranges in the range collection. The enumeration starts with the first item in the range, then the second is incremented etc. The specified range collection is checked whether there is a duplicate dimension included. If this is the case, an InvalidArgument-exception is thrown. Also, if the start index is behind the end index, an InvalidArgument-exception is thrown.
